# Keyboard Shortcuts

Keybindings are active when the canvas is focused and no text input field is active.

## Selection

| Key | Action |
|-----|--------|
| `Ctrl+A` | Select all nodes |
| `Escape` | Deselect all nodes (or cancel connection drop menu if open) |

## Editing

| Key | Action |
|-----|--------|
| `Delete` | Delete selected nodes |
| `Ctrl+D` | Duplicate selected nodes |

## Navigation

| Key | Action |
|-----|--------|
| `F` | Frame selection (centers view on selected nodes, or all nodes if none selected) |
| `Home` | Reset view (zoom to 1.0, offset to origin) |

## Panels

| Key | Action |
|-----|--------|
| `Tab` | Toggle script editor panel |

## File Operations

| Key | Action |
|-----|--------|
| `Ctrl+S` | Open save gates dialog |
| `Ctrl+O` | Open load gates dialog |

## Quick Actions

| Key | Action |
|-----|--------|
| `Space` | Open quick spawn menu at cursor position |
